Synchronisation tableau → carte

Configurez Sélectionner par (panneau latéral Grist → Données) sur le widget carte en choisissant le tableau de données. Clic sur une ligne → la carte zoome sur l'entité.

⚠ La direction inverse (clic carte → sélection dans le tableau) n'est pas supportée par l'API Grist. Clic carte = surbrillance visuelle sur la carte uniquement.

Données

Colonne géométrie (GeoJSON) ou paire latitude/longitude utilisée pour afficher les entités. Colonne étiquette affichée au survol. Couleurs et opacités des entités normales et sélectionnées.

Carte

Fond de carte, couches WMS additionnelles, centre et zoom par défaut. Ajuster à l'étendue : la carte se recadre automatiquement sur les données à chaque chargement.

Partage

URL de base sViewer (pour le lien de partage), URL de l'API Grist (instances auto-hébergées), intégration geOrchestra (catalogue CSW, proxy WMS).

Appliquer / Annuler

Appliquer reconstruit la carte avec les nouveaux paramètres. Pour les conserver au rechargement, cliquez ensuite sur Enregistrer dans la barre Grist (en dehors du widget). Annuler ferme sans modifier.

Exporter / Importer

Exporter copie la configuration dans le presse-papiers (JSON). Importer charge un JSON précédemment exporté pour dupliquer la configuration sur un autre widget.
