Configurer ce widget

Sélectionnez Complet dans Accès pour pouvoir modifier les géométries depuis la carte. En lecture seule, les boutons de saisie sont masqués.

Cliquez sur une ligne du tableau pour centrer la carte sur l'entité correspondante.

Le bouton Partager de sViewer génère un lien autonome vers la carte — le document Grist doit être accessible publiquement.

Onglet Données

Définit comment la colonne géométrie est lue (GeoJSON, WKT, coordonnées…) et l'apparence des données sur la carte.

Onglet Avancé

Données WMS et catalogue : identifiants à saisir tels que configurés dans sViewer. URLs : à renseigner uniquement pour une instance auto-hébergée de Grist ou geOrchestra. Configuration JSON : permet de copier la configuration d'un widget vers un autre.
