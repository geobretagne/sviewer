CSV

extension

Charge un fichier CSV distant comme données vectorielles — géométrie auto-détectée depuis les noms de colonnes

csvimportdonnées vectorielles
Paramètres URL 6
ParamètreExempleDescription
_format _format=csv Forcer l'activation pour URLs sans extension .csv csv
_geommode _geommode=latlon Mode de lecture géométrie geojson · latlon · latlon_str · lonlat_str · wkt
_geomcol _geomcol=geom Nom de colonne géométrie JSON ou WKT (override auto-détection)
_collat _collat=lat Nom de colonne latitude
_collon _collon=lng Nom de colonne longitude
_labelcol _labelcol=nom Colonne utilisée comme étiquette sur la carte
  • CSV avec colonnes lat/lon — Auto-détection colonnes latitude/longitude
  • CSV sans extension .csv — Forcer l'adaptateur CSV pour URLs sans extension
v1.0.0 sViewer ≥ 0.10.0 fphg
