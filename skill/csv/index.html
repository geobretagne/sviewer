sViewer — CSV skill

This skill loads remote CSV files as map features. It runs in the background — there is no interactive interface.

Usage: add adapters: ['csv'] to your customConfig.js, then open sViewer with:

?geojson=https://your-server/your-file.csv

CSV files must be UTF-8, CORS-enabled, and have latitude/longitude columns (or a geometry GeoJSON column).
